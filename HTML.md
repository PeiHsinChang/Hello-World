# HTML

一個HTML的**基本架構**

```
<!DOCTYPE html>
<html>
  <head>
    <!--宣告編碼為UTF-8-->
    <meta charset="UTF-8" >
    <title></title>
  </head>
  <body>
  <h1>Hello World</h1>  
  </body>
</html>
```

HTML有基本的標籤

```<h1>...<h6> ```的主題標籤最大的字體為```<h1> ```最小為```<h6>``` 
 
```<table>```標籤則有```<tr> <td>```標籤
```<td>```則被包在```<tr>```裡
以階層排序則為 ```<table>``` -> ```<tr>``` -> ```<td>```

另外常用的標籤有```<div> <span> <p> ```及```<a>```
